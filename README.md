# camera-prep — релізи

Публічний канал релізів **camera-prep** — ПЗ провіжинінгу та калібрування камер
**Savgood SG-UCM8010NKL** (Windows). Тут лежать завантажувані Windows-бандли і `latest.json`;
вихідний код — у приватному репозиторії.

## Завантаження

У розділі [**Releases**](https://github.com/Milvoos/camera-releases/releases) візьми останній
`camera-prep-vX.Y.Z-win64.zip`.

## Встановлення й запуск

1. Розпакуй архів у будь-яку теку (напр. `C:\camera-prep\`). Усередині буде тека `camera-prep\`
   з `camera-prep.exe` та залежностями. **Не запускай `.exe` прямо з архіву** — Windows розпакує
   його в read-only temp, і застосунок не зможе писати дані.
2. Запусти `camera-prep\camera-prep.exe`.
   - Бандл **без цифрового підпису**, тож Windows SmartScreen може попередити → «Докладніше» →
     «Виконати попри це».
   - При першому запуску поруч із `camera-prep.exe` автоматично створяться порожні теки
     `data\profiles\` і `data\firmware\`.

## Перший запуск — налаштування (обов'язково)

Облікові дані та IP **не входять в архів** — їх вводить інженер локально:

1. Відкрий **«Налаштування…»**.
2. Введи: **IP камери** (заводський) і **заводський** логін/пароль; **цільовий IP**, **шлюз** і
   **цільові** логін/пароль (які виставить провіжинінг).
3. Збережи. Значення лягають у **локальний** `%APPDATA%\camera-prep\settings.json` (на цій машині,
   не в архіві). Наступні запуски їх пам'ятають.

Доки поля порожні, кнопка **«Старт» неактивна** й показується підказка, які саме дані заповнити.

## Куди класти профілі та прошивки

Поруч із `camera-prep.exe` є тека `data\`:

| Тека | Вміст | Розширення |
|---|---|---|
| `data\profiles\` | профілі конфігурації камери (для імпорту) | `*.backup` |
| `data\firmware\` | файли прошивок (опційно) | `*.bin` |

Поклади свої файли у відповідні теки й **перезапусти застосунок** (або відкрий «Налаштування») —
селектори профілю/прошивки підхоплять їх. Без прошивки крок оновлення прошивки просто пропускається;
без профілю провіжинінг не стартує.

## Вимоги

- **Windows 64-bit** з інтернетом.
- **Ethernet** до камери.
- **USB-TTL** перехідник (serial) для VISCA-керування.

## Оновлення

Наразі — завантаж новий `camera-prep-vX.Y.Z-win64.zip` вручну й розпакуй поверх (тека `data\`
з твоїми профілями/прошивками та `settings.json` у `%APPDATA%` не чіпаються). Вбудований
авто-апдейтер — у планах (Фаза 3).

## Технічне

Релізи публікуються автоматично з CI при пуші тега `vX.Y.Z` у приватному репозиторії.
Канал оновлень (для майбутнього апдейтера): `latest.json` у корені `main` —
`https://raw.githubusercontent.com/Milvoos/camera-releases/main/latest.json`
(поля: `version`, `url`, `notes`, `published`).
