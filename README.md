# camera-releases

Публічний канал релізів **camera-prep** — ПЗ провіжинінгу та калібрування камер
Savgood SG-UCM8010NKL. Тут лежать лише завантажувані Windows-бандли (GitHub Releases)
і `latest.json`; вихідний код — у приватному репозиторії.

- **Завантажити останню версію:** див. розділ *Releases*.
- **Канал оновлень** (для вбудованого апдейтера Фази 3): `latest.json` у корені `main` —
  `https://raw.githubusercontent.com/Milvoos/camera-releases/main/latest.json`
  (поля: `version`, `url`, `notes`, `published`).

Релізи публікуються автоматично з CI при пуші тега `vX.Y.Z` у приватному репозиторії.
